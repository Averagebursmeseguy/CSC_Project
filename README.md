# Welcome to AstroNav
 Welcome aboard Navigator. Your guidance shall lead us to victory. Or at least not get slammed into the side of an asteroid. idk.

## What is the game about?
 Comp Sci project, indie game and endless time sink all in one. This game (hopefully) teaches you how to think in code. 

## How does it do that?
 The game is simple. There's a map, there's a player character and there's an end point. Basically you write a simplified syntax of code to make the player character get from the start to the end point. You could, make them move by just individual commands. Or you could make functions and poll for inputs so the player character dynamically responds to the map and its location.
